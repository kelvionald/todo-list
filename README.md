# ToDo List

## Требования к ПО

- Apache 2.4
- PHP 7.4
- MariaDB 10.3

## Запуск приложения

- Создать базу в MariaDB
- Скопировать файл .env.example -> .env
- Настроить .env
- Запускаем ```composer install```
- Запускаем ```php -t public -S localhost:8005```
- Переходим по ссылке в браузере http://localhost:8005/
